# Introduction

## Usage

```javascript
var tta = new TextTypingAnimation(document.getElementById('tta'));
tta
  .go({
    text: 'Hello World!',
    duration: 500
  })
  .go({
    text: '\nText Typing Animation',
    delay: 100
  });
```
